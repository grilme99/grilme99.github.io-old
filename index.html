  • RoCheck

    • GitHub
    • -
    • NPM

    RoCheck is a JavaScript module that allows you to get the real IP address of any running Roblox game server. This enables you to cross-compare the IP address of any API requests made to your server to the actual Roblox IP so you can be sure that the request is authentic.
